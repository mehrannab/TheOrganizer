For at køre projektet på en emulator, kræves det at Android Studio er installeret. Der er nemlig begrænset funktionalitet, når projektet køres på web.

1. For at få projektet til at virke, skal man køre: "npm install --force"

2. Herefter køres Expo start(hvis man ikke har det, skal den downloades via "npm install --global expo-cli" også derefter køres Expo start)

NB. Det kan være at Expo start skal køres to gange før projektet kommer frem.


OBS. I koden ses det, at der er en komponent ved navn InformationComponent, hvorved dette kaldes UserInformationComponent igennem hele rapporten og bilaget.
